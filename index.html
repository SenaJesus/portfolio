Projects

MEUSISU.com

THE PROJECT

I, along with two friends, developed the first website in Brazil to provide information on daily partial cutoff scores for the SISU.


The project involved creating a clean and fast interface to allow students preparing for the ENEM to easily analyze their chances of getting into their dream courses.


Since the government does not provide such information, we embarked on a parallel web scraping project that was able to daily collect over 160000 data points for all available courses. We then organized this data in our database so that the website could automatically display it.


Throughout the project, I honed my project planning skills, starting with designing the interface using Figma.


We also collaborated with various professors from renowned test preparation courses in Brazil to gather feedback and find the best approach for creating the website.

Automatic Spotted

THE PROJECT

I developed a project that automated the entire manual process of creating Instagram posts for a university entity.


Previously, when a student submitted a "letter" through a Google form, the account administrators had to manually create the artwork for the post. With the goal of speeding up the process, the project involved a system hosted on a server that automatically collected these letters from the form, generated the corresponding artwork, and requested confirmation from moderators to verify if the publication was feasible.


By doing so, we significantly reduced the time required to create posts and ensured greater efficiency in the Instagram publishing process.
